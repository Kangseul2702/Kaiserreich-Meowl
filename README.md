# 먀울 모드 (Meowl Mod)

우리는 먀울을 지지합니다.

# 모드 소개

이 프로젝트는 카이저라이히(Kaiserreich) 모드를 기반으로 제작된 서브 모드입니다.
기존 카이저라이히의 세계관을 바탕으로 우리만의 독특한 재미를 더하는 것을 목표로 합니다.

# 개발 방향성

"고증? 말이 되는 정도면 OK!"

먀울 모드의 제1원칙은 재미입니다.
복잡하고 엄격한 역사적 고증보다는 플레이어가 즐거움을 느낄 수 있는 방향을 추구합니다. 물론, 최소한의 개연성을 위해 '말이 되는 수준'의 설정은 유지하려고 노력합니다.

# 기여 및 초대 관련 안내

이 저장소(또는 프로젝트)에 초대를 받으셨나요?

부담 갖지 마세요: 초대를 받았다고 해서 반드시 무언가를 개발하거나 기여해야 하는 것은 아닙니다.

자유로운 참여: 우리는 불특정 다수에게 자유롭게 초대를 보냈습니다.

관전 환영: 들어오셔서 아무것도 하지 않고 구경만 하셔도 전혀 상관없습니다.
